# Calibration of AI-Based Automated Paediatric Dental Images

This repository contains the web application workspace wrapper developed for **Team-14 Internship I** at **Sri Ramachandra Faculty of Engineering and Technology**.

## 🔬 Project Overview
The platform provides a high-contrast, professional user interface to interact with a deep learning diagnostics engine designed for early childhood (ages 0–6) dental structures. 

## 🚀 How to Run the App Locally
1. Download or clone the `DentaliaDental.html` file.
2. Double-click the file to open it in any modern web browser.
3. The interface will automatically connect securely to our live model backend hosted on Hugging Face Spaces.
